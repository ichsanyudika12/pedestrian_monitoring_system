## Requirements

- OpenCV 4.x + OpenCV Contrib

- ZED SDK: https://www.stereolabs.com/developers/

- YOLOv8 model (models.onnx)

- C++17 compatible compiler

- Make sure your camera is properly connected and calibrated

### Build & Run

1. Clone Repository

       git clone https://github.com/ichsanyudika12/pedestrian_monitoring_System.git
       cd pedestrian_monitoring_System

2. Build
   
       mkdir build
       cd build
       cmake ..
       make

3. Run

       ./realtime
