# Requirements

- OpenCV 4.x + OpenCV Contrib
  
- ZED SDK: https://www.stereolabs.com/developers/
  
- YOLOv8 model (models.onnx)
  
- C++17 compatible compiler

- Make sure your camera is properly connected and calibrated before running the system

## Build & Run

- git clone https://github.com/ichsanyudika12/Pedestrian-Monitoring-System.git

- cd Pedestrian-Monitoring-System

- mkdir build

- cd build

- cmake ..

- make

- ./realtime
